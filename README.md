### How to use:
    1. Download this project from GitHub
    2. show hidden files and folder and then remove '.git' (must need) ***
    3. npm install
    4. npm run tw:build
    5. npm run tw:watch
    (now do your work with html, js, images etc)
    6. open new terminal and run 'npm run tw:prod' for production
